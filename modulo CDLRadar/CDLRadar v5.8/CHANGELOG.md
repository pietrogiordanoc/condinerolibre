# Changelog

Todos los cambios notables del proyecto serán documentados aquí.

El formato está basado en [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
y este proyecto usa [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [5.7.0] - 2026-03-13

### Added
- Sistema de auto-ordenamiento de señales "NOW" por powerScore
- Las señales nuevas se alinean automáticamente en la parte superior
- Exportación de GlobalAnalysisCache para ordenamiento inteligente

### Initial Features
- Análisis multi-temporal (4H, 1H, 15M, 5M)
- Sistema Orchestra V4 con 3 estrategias (SMC, SqzMom, MACD)
- Power Score de 0-100 puntos
- Trade setup dinámico con TP basado en ATR
- Alertas de audio para señales de entrada/salida
- Integración con Supabase para datos históricos
- Modal TradingView para gráficos interactivos
- Radar financiero con visualización circular
- Sistema de bookmarks en localStorage
- Detección de mercado abierto/cerrado
- Soporte para Forex, Índices, Acciones, Crypto y Commodities
